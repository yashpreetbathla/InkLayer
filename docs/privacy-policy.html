InkLayer for Chrome

No data collected

Privacy Policy

InkLayer is a Chrome extension that lets you draw annotations on any webpage and capture the result as an image. This policy explains what the extension does — and does not do — with your data.

Last updated: February 2026

The short version

InkLayer collects no personal data, stores nothing remotely, and never transmits any information about you or the pages you visit to any server. Everything happens entirely inside your browser.

What we do not collect

  • No browsing history or page content
  • No personal information of any kind
  • No analytics, telemetry, or usage data
  • No cookies or persistent identifiers
  • No screenshots or annotations are uploaded anywhere

Permissions and why they are needed

InkLayer requests the minimum permissions required for its features to work. Here is exactly what each permission is used for:

Permission Why it's needed
activeTab Injects the annotation canvas into the page you are viewing when you activate the extension. Only fires on explicit user action.
tabs Reads the active tab's window ID so the screenshot API can capture the visible viewport for Save and Copy.
scripting Injects the annotation scripts into tabs that were already open before the extension was installed, so you don't need to reload the page.
clipboardWrite Writes the composited screenshot + annotation image to your clipboard when you click the Copy button. The clipboard is never read.
host_permissions Allows the annotation canvas to be injected on any website you choose to annotate. No page content is read or transmitted.

Remote code

InkLayer does not load or execute any remote code. All JavaScript is bundled locally within the extension package. No external scripts, eval(), or dynamic code execution is used at any point.

Data storage

InkLayer stores nothing. No data is written to chrome.storage, localStorage, IndexedDB, or any remote server. When you close the tab or press Escape, all annotations are discarded entirely.

Third-party services

InkLayer does not integrate with any third-party services, analytics platforms, advertising networks, or APIs. No data is shared with anyone for any reason.

Children's privacy

Because InkLayer collects no data from anyone, it is safe for users of all ages and complies with COPPA and similar regulations.

Changes to this policy

If this policy ever changes, the updated version will be posted at this URL with a new "Last updated" date. Given that InkLayer collects no data, meaningful changes are unlikely.
